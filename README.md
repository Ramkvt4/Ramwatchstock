# Stocks Watchlist

A simple web application to track stock prices in real-time using the Alpha Vantage API.

## Features

- Display a list of stocks with their symbols and real-time prices.
- Fetch stock prices using the Alpha Vantage API.
- Responsive design for various devices.

## Getting Started

### Prerequisites

- Web browser
- Alpha Vantage API key (sign up [here](https://www.alphavantage.co/))
